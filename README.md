## Project Description
This project is a responsive multi-page website built using HTML and CSS. It demonstrates structured layout design, modern styling techniques, and the use of Flexbox and CSS Grid for creating responsive web pages.

## Technologies Used
- HTML5
- CSS3
- Google Fonts
- Flexbox
- CSS Grid

## Features
- Semantic HTML structure (header, main, section, footer)
- Responsive multi-page layout (Home, About, Projects, Contact)
- Flexbox navigation bar
- CSS Grid project gallery
- Styled contact form
- Consistent and modern design

## How to Run
1. Clone this repository
2. Open `index.html` in your browser

## Lessons Learned
I learned how to create responsive layouts using Flexbox and CSS Grid, how to structure multi-page websites, and how to apply consistent styling across different pages.

## Challenges Faced
I faced challenges with responsive design, especially in using Flexbox and Grid layouts, but improved by practicing layout structuring and using media queries.
